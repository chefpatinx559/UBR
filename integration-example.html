Exemple d'intégration des modules JavaScript

Comment intégrer les modules JavaScript

Pour intégrer les modules JavaScript extraits dans vos pages HTML existantes, suivez ces étapes :

  1. Ajoutez d'abord le module utils.js qui contient les fonctions utilitaires communes.
  2. Ajoutez ensuite les modules spécifiques dont vous avez besoin (forms.js, user-management.js, testimonials.js, reports.js).
  3. Enfin, ajoutez le fichier app.js qui initialise les fonctionnalités en fonction de la page.

Exemple de code à ajouter avant la fermeture de la balise body

<!-- Modules JavaScript -->
<script src="assets/js/modules/utils.js"></script>

<!-- Modules spécifiques (ajoutez uniquement ceux dont vous avez besoin) -->
<script src="assets/js/modules/forms.js"></script>
<script src="assets/js/modules/user-management.js"></script>
<script src="assets/js/modules/testimonials.js"></script>
<script src="assets/js/modules/reports.js"></script>

<!-- Initialisation de l'application -->
<script src="assets/js/app.js"></script>

<!-- Script principal existant (conservez-le si nécessaire) -->
<script src="assets/js/main.js"></script>

Modules disponibles et leurs fonctionnalités

utils.js

Fonctions utilitaires communes à toutes les pages

  • Affichage de notifications toast
  • Compteurs de caractères pour les zones de texte
  • Animations au défilement
  • Validation de formulaires
  • Validation d'email
  • Formatage de numéros de téléphone

forms.js

Gestion des formulaires

  • Soumission de formulaire avec animation
  • Réinitialisation de formulaire
  • Auto-suggestion basée sur le type de sélection
  • Auto-complétion du sujet basée sur les mots-clés
  • Détection d'urgence basée sur les mots-clés
  • Sauvegarde automatique des brouillons
  • Confirmation avant fermeture de page

user-management.js

Gestion des utilisateurs

  • Gestion des utilisateurs bloqués
  • Filtrage et recherche d'utilisateurs
  • Exportation de la liste des utilisateurs bloqués
  • Gestion des photos de profil

testimonials.js

Gestion des témoignages

  • Système de notation par étoiles
  • Modal de témoignage
  • Formulaire de témoignage
  • Filtrage des témoignages

reports.js

Gestion des signalements

  • Système de signalement
  • Pré-sélection du type de rapport
  • Suggestions automatiques
  • Soumission de signalement avec délai variable
