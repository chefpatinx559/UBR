Politique de confidentialité

Dernière mise à jour : 15 janvier 2025

1. Collecte des données personnelles

UBR collecte les données suivantes :

  • Données d'identification : nom, prénom, âge, photos
  • Données de contact : adresse email, numéro de téléphone
  • Données spirituelles : dénomination, pratiques religieuses, valeurs
  • Préférences : critères de recherche, localisation
  • Données d'usage : activité sur la plateforme, messages

2. Utilisation des données

Vos données sont utilisées pour :

  • Créer et gérer votre profil
  • Proposer des profils compatibles
  • Faciliter les échanges entre membres
  • Fournir un accompagnement personnalisé
  • Améliorer nos services
  • Assurer la sécurité de la plateforme

3. Partage des données

UBR ne vend jamais vos données personnelles. Nous ne partageons vos informations qu'avec :

  • Les autres membres (uniquement les informations de profil)
  • Nos prestataires techniques (hébergement, paiement) sous contrat de confidentialité
  • Les autorités légales si requis par la loi

4. Sécurité des données

Nous mettons en place des mesures de sécurité avancées :

  • Chiffrement SSL pour toutes les communications
  • Stockage sécurisé sur serveurs européens
  • Accès restreint aux données personnelles
  • Surveillance continue des activités suspectes
  • Sauvegardes régulières et sécurisées

5. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : consulter vos données personnelles
  • Droit de rectification : corriger vos informations
  • Droit à l'effacement : supprimer votre compte et vos données
  • Droit à la portabilité : récupérer vos données
  • Droit d'opposition : vous opposer au traitement

6. Conservation des données

Nous conservons vos données :

  • Tant que votre compte est actif
  • Jusqu'à 3 ans après résiliation pour les données de facturation
  • Les messages sont supprimés après 2 ans d'inactivité
  • Suppression complète sous 30 jours après demande

7. Cookies

UBR utilise des cookies pour améliorer votre expérience. Vous pouvez les désactiver dans votre navigateur, mais cela peut affecter certaines fonctionnalités.

8. Contact

Pour toute question concernant vos données personnelles, contactez notre délégué à la protection des données : dpo@ubr-rencontres.com
